# Data Visualization and Exploratory Data Analysis

## Problem Statement

The objective of this task is to perform Exploratory Data Analysis (EDA) and create meaningful visualizations to better understand the dataset. The goal is to identify patterns, distributions, feature relationships, and correlations that can assist in data-driven decision-making.

 ## Dataset Details

* Took dataset from kaggle
* The dataset consists of multiple numerical features along with a target category.
* It was loaded using the Pandas library.
* The data was inspected by viewing sample records and checking its structure, data types, and completeness before analysis.

---

## Approach

The following steps were performed:

1. Imported the dataset using Pandas.
2. Inspected the dataset using head() and info().
3. Created a Bar Chart to visualize category distribution.
4. Generated a **Histogram** to observe feature distribution.
5. Built a **Scatter Plot** to analyze relationships between two numerical features.
6. Used a **Pair Plot** to compare all numerical features.
7. Created a **Correlation Heatmap** to identify relationships among numerical variables using Pearson correlation.

##Tools & Libraries**

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

 ## Results

* Successfully performed exploratory data analysis on the dataset.
* Visualized the distribution of categories and numerical features.
* Identified relationships between different variables through scatter plots and pair plots.
* Analyzed feature correlations using a heatmap.
* Improved understanding of data visualization techniques and EDA using Python.
